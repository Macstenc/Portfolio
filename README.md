# Portfolio

To jest statyczne portfolio frontendowe Macieja Stencla.

## Jak wejsc lokalnie

Masz 2 proste opcje:

1. Otworz plik `index.html` bezposrednio w przegladarce.
2. Uruchom lokalny serwer w katalogu projektu:

```powershell
python -m http.server 5500
```

Potem wejdz na:

```text
http://localhost:5500
```

## Jak wejsc online

Portfolio po publikacji na GitHub Pages jest dostepne pod adresem:

```text
https://macstenc.github.io/Portfolio/
```

## Najwazniejsze pliki

- strona glowna: `index.html`
- dane i tresci: `scripts/data.js`
- style: `styles/main.css`
- skrypty: `scripts/main.js`
